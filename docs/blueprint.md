# **App Name**: Rollout HQ

## Core Features:

- Project Creation: Create projects (songs, tour dates, etc.)
- Project Overview: List/View all created projects
- Project Health Tracking: Set health status indicators on projects
- Target Date Assignment: Set target dates for project completion.
- AI Project Name Generation: Generate project name ideas based on keywords using an AI tool.

## Style Guidelines:

- Primary color: Dark Indigo (#4B0082) to evoke creativity and focus.
- Background color: Very dark gray (#1E1E1E), near-black for a professional, modern feel.
- Accent color: Gold (#FFD700) for highlighting key elements and CTAs.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text.
- Use simple, minimalist icons to represent project types.
- Clean, tabular layout for displaying project information.
- Subtle transition animations to improve UX when navigating.
